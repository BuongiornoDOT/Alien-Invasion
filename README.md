# Alien-Invasion
Классический аркадный шутер на PyGame 
[Alien Invasion]
Аркада где вы играете за космический корабль и отстреливаете пришельцев
Цель игры: [набить как моно больше очков]
Управление:
Вперед: W
Назад: S
Вправо: D
Влево: A
Стрельба: Space
Требуеться PyGame и Python 3.9 или новее(разрабатывалась на 3.12.7):
